Frequently Asked Questions about %@ v%@

What is Yarn?

Yarn is a free editor and player for Twine games specially created for mobile devices.

What is Twine and what type of games are considered Twine games?

Twine is an application for, but not limited to, creating choose your own adventure type games. Twine was originally created by Chris Klimas in 2009. To know more about Twine visit http://twinery.org.

Yarn is compatible with what version of Twine?

The current version of Yarn is compatible with Twine version %@.

Where can I find the source code for Yarn?

The source code is freely available and currently hosted at Github and is maintained by me, Mark Jundo P. Documento. If you want to contribute to its development, please feel free to tinker with the code or drop me a message at my mail.

What is the license for Yarn?

Original code for Yarn is released under the MIT license. Because it also bundles the (unmodified) story and proofing formats used by Twine 2, it is also dually licensed under GPLv3 in order to provide compatibilty with the Twine 2 license.
